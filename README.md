# Image-carousel
This **Image Caraousel** project is built with **HTML, CSS and JavaScript.**

---

## Features
<ul>
  <li>Navigation controls: Arrows for "next" and "previous" buttons.
</li>
  <li>
   Indicator Dots: Small dots at the bottom that show the total number of slides and which one is currently active.
  </li>
  <li>
    Auto-Sliding: Images transition every few seconds.
  </li>
</ul>

---

# TechStack
<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ul>
